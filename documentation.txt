# VALIDAÇÃO

Como um bom ORM ele deve manter a integridade do banco de dados logo ele antes de fazer qualquer alteração 
deve se verificar: 

     * A quantidade de colunas devem ser as mesmas (pegar o primeiro registro como referencia)
     * Se for inserir verificar se o valor antigo é o mesmo (pegar o primeiro registro como referencia)

Logo não tem como adicionar uma coluna a mais, já que antes desta operação ele faz a validação dos campos
de dados.
